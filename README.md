一.DatePicker组件
- 基于慕课网上DatePicker组件开发的课程
- 在老师的基础上进行改造 
  - 通过操作dom的方式，而不是全部都写在html字符串中
  - 解决了点击以后改变年份会出错的问题
  - 增加了鼠标悬停和选中的效果
  - 前一个月的日期显示为灰色
  
- 轮播图组件
  - 可以通过传入图片数量来控制
